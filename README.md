# Movie-Searcher

Search movies out of a movie database with few keywords.

Created using 

              OMDb API
              
              Node.js 
              
              Express.js
              
              HTML/CSS
              
              Javascript
              
              EJS

# Techniques 

              - RESTful routing.

              - JSON parsing.

              - Varius NPM modules.

              - EJS: Embedded Javascript.

              - Code reuse: custom assets and partials.
